Asistente Virtual — Resumen de Mejoras

Sesión de desarrollo · 21 de marzo de 2026

⚡
Estado inicial vs. estado actual
Antes Chatbot básico que solo respondía sobre Western Union y Fondos de Inversión. Sin frontend propio, sin documentos, sin formularios.
Ahora Asistente con 3 servicios, RAG con PDFs, 3 formularios de destino, interfaz de chat completa y detección inteligente de intención.
🌐
Nuevo servicio: Comercio Exterior
  • Se agregó un tercer servicio al chatbot: Soluciones de Comercio Internacional en alianza con Convera Nuevo
  • El bot carga automáticamente al arrancar 3 PDFs (GlobalPay, Soluciones Comex, Infografía) y usa su contenido para responder con precisión Nuevo
  • Detección por palabras clave: importación, exportación, GlobalPay, Convera, cobertura cambiaria, forward, entre otras Nuevo
📋
Formularios de destino
  • formulario-western-union.html — Para clientes que quieren enviar o cobrar un giro Nuevo
  • formulario-fondos.html — Para clientes interesados en invertir, con selector de perfil de riesgo y objetivo Nuevo
  • formulario-comercio-exterior.html — Para empresas que necesitan pagos internacionales o coberturas Nuevo
  • El bot envía el link al formulario correcto cuando identifica la necesidad del cliente, en vez de mostrar marcadores de texto Mejorado
💬
Interfaz de chat (nueva)
  • index.html — Página de chat completa servida desde el mismo servidor Nuevo
  • 3 botones de acceso rápido al inicio: Invertir / Western Union / Comercio Exterior Nuevo
  • Indicador de escritura animado (3 puntos) mientras el bot responde Nuevo
  • Conversión de Markdown a HTML: los **asteriscos** se renderizan como negritas, los saltos de línea se respetan Corregido
  • Historial de conversación completo enviado al API para que el bot recuerde el contexto Nuevo
  • Enter para enviar, Shift+Enter para salto de línea Nuevo
🖥️
Servidor
  • El servidor ahora sirve archivos estáticos (.html, .css, .js, imágenes) además del endpoint /chat Mejorado
  • Se corrigió la versión de pdf-parse (de v2.4.5 a v1.1.1) para que la lectura de PDFs funcione correctamente Corregido
✨
Funcionalidades adicionales
  • Cuando el cliente pregunta por la oficina más cercana, el bot responde con link a la página de oficinas y a Google Maps ("Western Union - Acciones y Valores") Nuevo
